第二次开会内容：
功能：
    定时：定时暂停
    识别：用什么模型、要什么数据
    用户输入：预处理
    保存/加载：迭代二
    
操作流程：
    1、双击exe
    2、配置窗口，配置完并确定
    3、关闭配置窗口、打开主窗口，主窗口根据配置文件的内容相应的改变
    4、开始笔输入，画图形
    5、画布纠正图形并显现出来（不封闭模型暂定static）
    6、画一个箭头，箭头根部与物体之间有一个阈值
    7、出现一个提供单次输入的框
    8、用户在框内输入参数，并确认参数输入
    9、重复操作8，直到参数输入完毕
    10、点击运行
    额外操作：暂停、计时定时、保存/加载
